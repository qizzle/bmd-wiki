## Usage

With this mod, you can generate images for free using https://prodia.com/.

**Base Model:** You can choose between SD and SDXL. SDXL is a more advanced version of SD, which means it might take longer to generate images but will produce higher-quality results.

**Model:** There are many models available for both SD and SDXL. You can only use SD models if you've selected SD as your base model, and the same applies to SDXL.

**Prompt (required):** Enter your prompt here. Whatever you type will be generated as an image.

**Negative Prompt:** Enter things you don't want in the image here. This helps avoid errors in the generated images. Use the example in the placeholder as a good general negative prompt.

**Store Image URL:** This is the URL where the generated image will be stored.

Options beyond these basics are for advanced users and are not included in this guide.
